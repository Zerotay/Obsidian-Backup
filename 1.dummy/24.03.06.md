# 깃 설정
- 깃 커밋 시 지라 이슈 번호 자동 삽입
- 파일 대소문자 구별
- 커밋 시 메시지 나오도록 수정
- 브랜치 이름 변경 시 바로 원격에도 반영되도록 수정
- 커밋 시 제목 제대로 지정하도록 설정

깃 훅 세팅
https://git-scm.com/book/ko/v2/Git%EB%A7%9E%EC%B6%A4-Git-Hooks

https://www.lesstif.com/lpt/linux-cut-93127007.html
